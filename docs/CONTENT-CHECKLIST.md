# IYKA-ARAM — Content & Assets Checklist

Everything below is what turns the landing page from "premium placeholder" into "finished."
You can either **send it all to me** (I'll drop it in), or edit it yourself in one file:
**`src/content/site.ts`**. Images go in the **`public/`** folder (e.g. `public/brand/hero.jpg`),
then reference the path in `site.ts` (e.g. `"/brand/hero.jpg"`).

---

## 1. Brand assets (highest impact)
- [ ] **Logo** — vector if possible (`.svg`), otherwise high-res `.png` with transparent background
- [ ] **Brand colours** — exact codes if you have them (we're currently using gold / forest-green / cream sampled from the logo)
- [ ] **Hero photo(s)** — 1–3 high-res scenic Meghalaya / centre shots (landscape, ≥ 2000px wide)

## 2. Business details
- [ ] Phone number (for calls)
- [ ] WhatsApp number
- [ ] Email address
- [ ] Full address + Google Maps link (for the location section/map)
- [ ] Social media links (Instagram / YouTube / Facebook / etc.)

## 3. Copy (text)
- [ ] **Mission / Our Story** — a paragraph or two in your own words (we have a strong draft you can tweak)
- [ ] **About** content — the longer story behind IYKA-ARAM

## 4. Team
For each doctor / therapist / yoga instructor / nutritionist:
- [ ] Name
- [ ] Role / title
- [ ] Short bio (1–2 lines)
- [ ] Photo (portrait, ideally square, ≥ 600px)

## 5. Services
- [ ] Confirm the 3 service lines + a one-line description of each
- [ ] Prices (for display)
- [ ] Durations (for consultations/sessions)

## 6. Products
For each product:
- [ ] Name
- [ ] Price
- [ ] Short description
- [ ] Photo

## 7. Social proof
- [ ] **Testimonials** — 3+ real quotes + who said them (first name + town is fine)
- [ ] **Reels / podcast / event** links or videos to feature
- [ ] **Gallery** photos (centre, therapies, yoga, wellness food — the "5-star display")

## 8. Recognition
- [ ] Official emblem images for **Ministry of AYUSH** and **Ministry of Education** (if available)

---

### Priority order if you're short on time
1. Logo + 1 hero photo + real contact details → biggest instant upgrade
2. Team names + photos
3. Testimonials
4. Everything else
